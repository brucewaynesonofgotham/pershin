Теория

Основополагающие этапы в работе с picture:
    
  1. Написать открывающий и закрывающий теги <picture></picture>
  2. Внутри этих тегов создайте элемент <source> для каждого медиа-запроса, который вы хотите запустить.
  3. Добавьте аттрибут media, содержащий информацию о признаке, по которому будет происходить выбор изображения
  4. Добавьте атрибут srcset с соответствующим именем изображения для загрузки.
  5. Добавьте дополнительные имена файлов в атрибут srcset, если вы хотите указать дескрипторы плотности пикселей для корректного отображения ретинных дисплеев
